    What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is when you add a single event listener to a parent element that handles events for all its child elements. This way, instead of adding listeners to each child, the parent listens for events and can respond when any child is interacted with. Itâ€™s useful because it improves performance, simplifies your code, and works well with elements that may be added later.

    How do you manipulate an elements attributes and styles using the Dom?

    You can modify an element's attributes and styles in the DOM like this: Attributes: Get: Use element.getAttribute('attrName') to read an attribute. Set: Use element.setAttribute('attrName', 'value') to change or add an attribute. Remove: Use element.removeAttribute('attrName') to delete it. Styles: Change a style: Use element.style.propertyName = 'value' (e.g., element.style.color = 'blue'). Set multiple styles: Use element.style.cssText = 'color: blue; background: yellow;'. This makes it easy to change how elements look and behave!
